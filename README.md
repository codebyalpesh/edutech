# EduTech

[![Demo](https://img.shields.io/badge/Demo-Live-blue)](https://codebyalpesh.github.io/edutech/)

**EduTech** is a modern, responsive, and interactive educational platform designed to make learning engaging and accessible for students and educators.

---

## 🔹 Demo
Check out the live demo here: [EduTech Live](https://codebyalpesh.github.io/edutech/)

---

## 🚀 Features
- Fully responsive UI for desktop and mobile  
- Interactive course and lesson modules  
- Quiz and assessment system  
- Student progress tracking and analytics  
- Modular and scalable frontend architecture  
- User-friendly interface with clean design  

---

## 🛠 Tech Stack
- **HTML5 & CSS3** – Semantic markup & responsive layouts  
- **JavaScript** – Interactive features and dynamic UI  
- **Vanilla JS / Framework-ready** – Easy to integrate with modern frameworks
